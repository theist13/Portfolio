About Game Development Projects

The game development projects undertaken within the university setting are primarily designed as study endeavors, aimed at providing students with a hands-on learning experience in various aspects of game development.


Stabilizer

About the Project

The thesis project "Stabilizer" presents a shooter game where players engage in missions to shoot down enemy robots. Set in a dystopian world, the game offers a straightforward yet action-packed experience.

Key Features:

  • First-Person Camera System: Provides a smooth and immersive first-person camera experience, allowing players to fully engage with the game environment and action.
  • AI Character Movement: Implements advanced AI pathfinding and behavior to create challenging and intelligent enemy robots. Utilizes root motion to ensure fluid and lifelike movement animations.
  • Intuitive Controls: Designed for easy-to-learn, hard-to-master controls that allow for precise aiming and shooting mechanics in a first-person perspective.
  • Mission-Based Gameplay: Players embark on various missions to shoot down enemy robots, each mission presenting unique challenges and objectives.
  • Learning and Testing: Created as an educational tool to explore and understand advanced Unity features, including AI animation techniques and first-person camera mechanics. Designed to facilitate learning how to develop an FPS game for desktop platforms.

"Stabilizer" aims to provide a comprehensive learning experience for those looking to delve into first-person shooter game development, while also offering a thrilling and engaging gameplay experience.

Untitle TPS Game

About the Project

This project is developed as a learning and testing ground for implementing a third-person view camera system using Cinemachine, alongside AI character movement utilizing root motion for realistic animations.

Key Features:

  • Third-Person Camera System: Integrates Cinemachine for smooth and dynamic camera control, providing an immersive experience.
  • AI Character Movement: Utilizes root motion to drive AI character animations, ensuring fluid and lifelike movement.
  • Endless Gameplay: The game has an endless point system where the player shoots to reach a high score.
  • Learning and Testing: Designed as an educational tool to explore and understand advanced Unity features, including Cinemachine and AI animation techniques, and to learn how to create a game in WebGL.
